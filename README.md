# مدونة فيزياء.تك (Physique.tech Blog)

مدونة تعليمية إخبارية تقنية تركز على تبسيط مجالات الفيزياء، البرمجة، والذكاء الاصطناعي وتقديم أحدث الأخبار والدورات التفاعلية.

## نظرة عامة (Overview)

تم بناء هذا المشروع ليكون منصة متكاملة و**متجاوبة 100% مع جميع الشاشات** لتبادل ومشاركة المعرفة. يعتمد الموقع على بنية HTML/CSS نظيفة باستخدام إطار عمل `Tailwind CSS`، ويستخدم الجافاسكريبت النقي (Vanilla JS) لبناء الواجهات التفاعلية الديناميكية دون الحاجة لإطارات عمل معقدة، مما يجعله خفيفًا وسريع الاستجابة.

## المميزات الرئيسية (Key Features)

- **مقالات حصرية:** قسم مخصص للمقالات مع عرض مستمر لأحدث اكتشافات الفيزياء وتطورات الذكاء الاصطناعي مع توثيق المراجع.
- **محرر مقالات مدمج (Markdown Editor):** إمكانية للمديرين لكتابة وتنسيق المقالات بسهولة عالية ونشرها مباشرة.
- **كورسات برمجة وتطوير:** بيئة تفاعلية لعرض دروس ولغات البرمجة الشائعة لتسهيل التعلم الذاتي.
- **مستودع رقمي (Hub):** يحتوي على أدوات وروابط لبرمجيات وتقنيات تخدم مجالات الموقع المختلفة.
- **الوضع الفاتح والداكن:** دعم أساسي مدمج للتحويل السلس بين الوضع الليلي والنهاري (Dark/Light Mode) ليناسب راحة العين.
- **محرك بحث عالمي (Global Search):** يمكن الوصول إليه في أي صفحة عبر أيقونة البحث أو باستخدام اختصار لوحة المفاتيح `Ctrl+K`. يستعرض النتائج بلمح البصر، مما يعزز سهولة **التنقل في الموقع كاملاً**.
- **مرونة في التصفح:** وجود زر الصعود لأعلى (Scroll to Top) وشريط تنقل مثبت (Sticky Navbar) لسهولة الحركة بين أقسام الموقع.
- **أزرار التنقل (Navigation Mode):** دعم التنقل السلس عبر قوائم الموبايل المصممة خصيصاً للقراءة والوصول السريع، والقدرة على الانتقال بين الدروس السابقة واللاحقة.

## بنية المشروع (Structure)

- `index.html`: الصفحة الرئيسية، تعرض أبرز المقالات مع التصنيفات وعلامات التبويب.
- `article.html`: واجهة عرض المقال المستقل. تقرأ من ملفات الماركداون وتحولها لنص منسق.
- `programming.html`: الواجهة المخصصة لاختيار الدورات التفاعلية في البرمجة وتكنولوجيا المعلومات.
- `lesson-view.html`: واجهة قراءة الدروس التفاعلية مع دعم التنقل المسبق والتالي.
- `hub.html`: قسم المستودع الرقمي والأدوات المفيدة.
- `editor.html`: واجهة إضافة وتحرير المقالات.
- `about.html` / `privacy.html`: صفحات معلومات الموقع والخصوصية.
- `articles.js` / `courses_*.js`: ملفات قواعد البيانات الستاتيكية (Static JS Data). تستخدم لحفظ قائمة المقالات والدروس قبل توفير خادم خلفي.
- `posts/`: مجلد يحتوي على جميع نصوص المقالات بصيغة Markdown (`.md`).
- `search.js` / `theme.js`: أدوات وبرمجيات الجافاسكريبت لإدارة البحث الشامل وتغيير السمة (المظهر الفاتح/الداكن).

## التعليمات التشغيلية (How to run)

نظرًا لأن التطبيق يستعمل وظيفة `fetch()` في لغة جافاسكريبت لجلب وقراءة الملفات النصية الماركداون لتوليد المحتوى بمرونة، يُفضل تشغيل المشروع باستخدام **Live Server** المتوفر في إضافات (VSCode) أو أي بيئة خادم محلي أخرى (Localhost Server) بدلاً من فتح الملفات مباشرة بالمتصفح، لكي يتسنى للتطبيق جلب البيانات بدون التعرض لقيود الـ (CORS).
